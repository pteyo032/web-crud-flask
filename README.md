# Application CRUD (Flask + SQLite)

## Description
Application web CRUD (Créer/Lire/Modifier/Supprimer) développée avec Flask et SQLite.
Le projet inclut une interface HTML ainsi qu’une API REST JSON.

## Fonctionnalités
- CRUD via interface web (HTML)
- CRUD via API REST JSON
- Validation serveur (titre requis, limites de longueur)
- Gestion d’erreurs 404 (HTML + JSON)
- Persistance des données via SQLite

## Technologies
- Python
- Flask
- SQLite
- HTML (Jinja2)
- CSS (minimal)

## Apperçu
http://127.0.0.1:5000
